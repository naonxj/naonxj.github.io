00. Network Topology

전체 네트워크 구성도 (GNS3)

GNS3를 이용하여 가상 네트워크 인프라를 구축하고, 방화벽 정책 설정, 라우터 이중화(HSRP), VPN 서버 구축, IDS 및 SIEM 시스템을 각각 개별적으로 구축하여 실습한 프로젝트입니다.

01. GNS3 Infrastructure Setup

GNS3 내 가상 장비를 활용하여 방화벽 정책과 라우터 이중화(HSRP)를 구성했습니다.

1. Firewall Rules (Cisco ASAv)

Rule 1: Firefox3에서 Inside 구역으로 http 접속 허용
Rule 1: Firefox3에서 Inside 구역으로 telnet 접속 허용
Rule 2: Webterm2에서 dmz 구역으로 telnet 접속 허용
Rule 3: PC1에서 Webterm1으로 ping 통신 허용
Rule 4: Webterm1에서 R1 라우터로 http 접속 허용
Rule 4: Webterm1에서 R1 라우터로 telnet 접속 허용
Rule 5: Webterm2에서 PC1으로 ping 통신 허용

2. HSRP Configuration (High Availability)

라우터 장애에 대비하여 HSRP(Host Standby Router Protocol)를 설정했습니다.

Active router: 주 경로로 동작하는 라우터 설정 (R4), Standby router: 대기 경로로 동작하는 라우터 설정 (R3) , Virtual Gateway : 192.168.50.254 설정

R4: HSRP Active 상태 확인 (Primary Gateway)
R3: HSRP Standby 상태 확인 (Failover 대기)

OpenVPN 서버 구축 및 설정 (pfsense)

1. Win10 -> OSSEC: http, ping 탐지
2. CentOS7 -> Win10: ping 탐지
