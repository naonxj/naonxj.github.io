2) Database Security: MariaDB SQL Challenge

비트 연산(Bitwise)을 활용한 개발자 기술 스택 조회 및 세부 권한 부여 관리 시나리오입니다.

SQL 문제: Front End 스킬 보유 개발자 조회 시나리오
1번 문제 해결: 비트 연산을 활용한 조건별 데이터 조회 쿼리 실행
2. Cdev: 로컬호스트 모든 권한 부여
3. Pydev: Skillcodes 테이블 Select/Create 권한 제어
4. Jdev: 원격 호스트(192.168.16.100) 대상 테이블 권한 부여

3) OS Hardening: Linux System Security

파일 권한 제어, 패스워드 복잡도 정책, 로그 분석 및 취약점 테스트를 통한 시스템 강화

권한 제어 및 보안 정책 (Sticky Bit & PW Policy)

Sticky Bit 설정: text.txt 파일에 대해 소유자 외 삭제 방지 보안 설정
종합 보안 설정:
1. 패스워드 복잡도(특수문자/숫자 포함 10자 이상, 만료일 90일)
2. wtmp/btmp 로그를 통한 접속 기록 및 로그인 실패 이력 분석
3. SetUID 설정을 이용한 권한 상승 시나리오 검증

Vulnerability Analysis (Buffer Overflow)

간단한 C 코드를 활용한 Buffer Overflow 취약점 실행 및 메모리 오염 확인 테스트
